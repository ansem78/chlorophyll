{{global.description}}

{{global.name}} is a featured CSS utilities framework which aims to help UI designers to quickly build websites an web-based apps from scratch. It can be used either standalone or easily integrated with other web frameworks, especially those that lacks utilities for layout, text alignment, positioning, etc. {{global.name}} does NOT make available any class to style colors, form inputs or buttons. It focuses on utility classes to format structure and any element on a page.

Features

  • Modularity: {{global.name}} follows a concept of features. Any major set of utility classes is part of a feature, and any feature can be configured independently, as well as enabled or disabled.
  • Highly configurable: You can configure only variable you need to override. And you can enable or disable features and their modules.
  • Mobile-first: {{global.name}} follows a modern mobile-first philosophy. However, its responsive utilities make it suitable for development for any device.
  • Lightweight: any @import rule (in CSS) or <link /> element (in HTML) do a HTTP call. To reduce the amount of these calls, {{global.name}} generates a single CSS file. The features system contributes to keep the CSS size as small as possible, based on features you really need.
  • Unopinionated: Any generated CSS rule is limited to style a single visual aspect for an element. Only the reset feature has opinionated rules, because it makes the appearance of any HTML element consistent across browsers.
  • Complete: {{global.name}} expresses CSS full potential. From simple formatting rules to animations and transformations. It also has utility classes to improve accessibility and readability, to truncate text and embed third-party media.
